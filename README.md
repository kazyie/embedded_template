# Embedded Template & TaskGen

> VS Code + WSL 上で動く、組み込みテンプレートと Python タスクジェネレータのサンプルプロジェクト

---

## 🎯 目的

- 組み込み風のマルチタスク C++ テンプレートを素早く作成  
- `taskgen.py` でタスク雛形（ヘッダー／ソース）を自動生成  

---

## ⚙️ 前提環境

- Windows 10/11 + WSL2（Ubuntu）  
- VS Code（Remote-WSL 拡張インストール済み）  
- Python 3.x（仮想環境推奨）  
- g++ (C++17)
